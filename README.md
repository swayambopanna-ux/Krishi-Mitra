# 🌾 Krishi Mitraan

An AI-powered Smart Agriculture Assistant that helps farmers with crop advisory, disease detection, irrigation decisions, and real-time sensor monitoring.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🌱 Crop Advisory System
- 📸 AI-Based Disease Detection (Leaf Analysis)
- 💧 Irrigation Recommendation System
- 🌡️ Real-Time Sensor Data (Moisture, Temperature, Humidity)
- 🌦️ Weather Information
- 🎤 Voice Assistant (Multi-language support)
- 🌍 Multi-language UI (English, Hindi, Kannada, Tamil, etc.)

---

## 🧠 Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- Multer (for image upload)
- Hugging Face API (AI Model)

### IoT
- NodeMCU (ESP8266)
- Soil Moisture Sensor
- DHT11 (Temperature & Humidity Sensor)
